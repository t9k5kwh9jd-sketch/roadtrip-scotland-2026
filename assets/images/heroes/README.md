# Hero-Bilder

`highlands-master.svg` ist eine verlustfreie 2400×1350-Vektorillustration und kann auf Retina-Displays nicht verpixeln.

Für echte Fotografie sollen ausschließlich eigene oder sauber lizenzierte Originaldateien eingesetzt werden. Empfohlen: 3000 px oder mehr, WebP/JPEG in hoher Qualität, keine hochskalierten Vorschaubilder.
